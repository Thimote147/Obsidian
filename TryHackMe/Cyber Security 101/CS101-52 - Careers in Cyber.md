# 💼 Careers in Cyber

> [!abstract] Summary
> Explore the main career paths in cybersecurity — SOC analyst to penetration tester to CISO. Understand skills, certifications, and learning roadmaps for each role.

**Path:** [[Cyber Security 101]] > Module 13 > Careers in Cyber

## 📖 Key Concepts

### Defensive / Blue Team Roles
| Role | Level |
|------|-------|
| SOC Analyst Tier 1 | Entry |
| SOC Analyst Tier 2/3 | Mid |
| Digital Forensics Analyst | Mid |
| Threat Intelligence Analyst | Mid |
| Security Engineer | Mid-Senior |
| CISO | Senior |

### Offensive / Red Team Roles
| Role | Level |
|------|-------|
| Penetration Tester | Mid |
| Red Team Operator | Senior |
| Bug Bounty Hunter | Variable |
| Vulnerability Researcher | Senior |

### Certifications by Path
**Entry:** CompTIA Security+ · Google Cybersecurity Cert · TryHackMe paths

**SOC/Blue Team:** CySA+ · BTL1 · Splunk Core Certified User

**Pen Testing:** eJPT (entry) · PNPT · **OSCP** (gold standard)

**Cloud:** AWS Security Specialty · Azure Security Engineer · CCSP

### Recommended Roadmap
```
1. TryHackMe Pre-Security         ✅ Done
2. TryHackMe Cyber Security 101   ← You are here
3. Choose direction: Offensive or Defensive
4. THM Jr. Penetration Tester  OR  SOC Level 1
5. HackTheBox Starting Point
6. eJPT / BTL1
7. Build home lab · get first job
8. OSCP / CySA+
```

> [!tip] SOC Analyst Tier 1 is the most accessible entry-level role — alert triage and log analysis are learnable skills with platforms like TryHackMe. It's the front door to many careers.

## 🔑 Key Takeaways
- Cyber careers: offensive (Red), defensive (Blue), governance (GRC)
- SOC Tier 1 is the most accessible entry point
- OSCP is the gold standard for penetration testing roles
- Practical skills (HTB, THM, home lab) matter as much as certs to employers
