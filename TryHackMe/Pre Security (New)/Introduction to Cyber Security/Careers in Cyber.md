## 📖 Career Roles

### 🗡️ Offensive Roles

#### Penetration Tester
- Tests systems, networks, and applications for vulnerabilities **with permission**
- Produces reports detailing findings and remediation steps
- Tools: [[Nmap]], [[Metasploit]], [[Burp Suite]], [[GoBuster]]
- Certifications: OSCP, CEH, eJPT

#### Red Teamer
- Simulates real-world **advanced persistent threat (APT)** attacks
- Goes beyond pen testing — tests detection and response capabilities
- Involves social engineering, physical intrusion, C2 infrastructure
- Requires deep knowledge of attack techniques (MITRE ATT&CK)

#### Bug Bounty Hunter
- Independently finds and reports vulnerabilities to companies for **monetary rewards**
- Platforms: HackerOne, Bugcrowd, Intigriti
- Flexible — can be done as a freelancer

---

### 🛡️ Defensive Roles

#### Security Analyst (SOC Analyst)
- Monitors [[SIEM]] alerts and investigates suspicious activity
- Triages incidents and escalates when needed
- Levels: L1 (triage) → L2 (investigation) → L3 (threat hunting)
- Certifications: CompTIA Security+, CySA+, BTL1

#### Incident Responder
- Responds to active security incidents
- Containment, eradication, and recovery
- Must act quickly under pressure
- Certifications: GCIH, GCFE

#### Digital Forensics Analyst
- Investigates attacks **after the fact**
- Recovers evidence from devices, memory, and network captures
- Supports legal proceedings
- Certifications: GCFE, GCFA, EnCE

#### Malware Analyst / Reverse Engineer
- Analyzes malicious code to understand behavior
- Uses disassemblers and debuggers (IDA Pro, Ghidra, x64dbg)
- Certifications: GREM

#### Security Engineer / Architect
- Builds and maintains security infrastructure
- Designs secure systems, networks, and policies
- Requires broad technical + business knowledge

#### CISO (Chief Information Security Officer)
- Executive-level role overseeing all security strategy
- Bridges technical security and business risk management

## 🔑 Key Takeaways
- Cybersecurity has diverse career paths — both technical and strategic
- Offensive and defensive skills complement each other
- Certifications help validate skills for employers
- Practical experience (CTFs, THM, HackTheBox) is equally important
